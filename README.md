A high-speed, neon-styled infinite runner game where you pilot a futuristic fighter, dodge enemies, and collect glowing diamonds.

## Features
- **Infinite Gameplay**: "No Box" dynamic world.
- **Neon Visuals**: Sleek vector graphics and glow effects.
- **Shop System**: Collect coins and buy new skins.
- **Persistence**: High scores and unlocks are saved automatically.

## How to Play
- **Desktop**: Arrow Keys or WASD to Move, SPACE to Shoot.
- **Mobile**: Touch and Drag to Steer, Tap Rocket Button to Shoot.

## How to Deploy on Vercel
1.  **Download** the provided `.zip` file.
2.  Extract it or keep it as is.
3.  Go to [Vercel.com](https://vercel.com) and log in.
4.  Click **"Add New"** > **"Project"**.
5.  Import the Git repository OR use the Vercel CLI to deploy this folder.
    *   *Alternative*: Simply drag and drop the folder contents if using a drag-and-drop static host like Netlify (Vercel usually requires Git or CLI).
    *   **Best Way**: Install Vercel CLI (`npm i -g vercel`), go to this folder in terminal, and type `vercel deploy`.

## Local Run
Simply open `index.html` in your browser.
